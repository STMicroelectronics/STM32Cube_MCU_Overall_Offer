# STM32Cube HAL2-based Embedded Software for STM32C5 series

## STM32Cube MCU Package

The STM32Cube MCU Package for STM32C5 series is a set of embedded software components including drivers, libraries and examples designed to help developers create applications for STM32C5 microcontrollers.

STM32Cube MCU Package                      | Repository
------------------------------------------ | -----------
STM32Cube MCU Package for STM32C5xx series | [Go to repository](https://github.com/STMicroelectronics/STM32CubeC5)
